# Sentiment Analysis on Social Media 🎓🤖

[![Python](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Analyze positive/negative sentiment from tweets or reviews using NLP and machine learning.

---

## ✨ Features
- 📄 Clean text preprocessing (lowercase, remove URLs/punctuation)
- 🧠 Logistic Regression model for sentiment classification
- 📊 Evaluation metrics (precision, recall, F1)
- 🎯 Predict sentiment for new text samples

---

## 📂 Dataset
This project uses the [Sentiment140 dataset](https://www.kaggle.com/datasets/kazanova/sentiment140), which contains 1.6 million tweets labeled for sentiment (positive/negative).  
Download the dataset and place `sentiment140.csv` in the project folder.
